# Todo App
A simple Todo app that helps you get stuff done.

## Getting started
### Prerequisites
- Node >=6.0

### Installation
Clone the project
```
git clone https://github.com/aparnadyadav/todo-app.git
```
Change into project's directory
```
cd todo-app
```
Start server
```
npm start
```

## Features
- Add a task
- Delete a task
- Mark a task as complete
- Filter tasks by status
